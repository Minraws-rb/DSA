# Day 9 - Revision (Active Recall)

## Topics Revised
- Time Complexity
- Arrays
- Linked List
- Stack
- Queue
- Recursion
- Searching

---

## 1. Time Complexity
- O(1): Constant
- O(n): Linear
- O(log n): Logarithmic
- O(n²): Quadratic

---

## 2. Arrays
- Contiguous memory
- Access → O(1)
- Insertion/Deletion → O(n)

---

## 3. Linked List
- Non-contiguous memory
- Access → O(n)
- Insertion/Deletion → O(1)

---

## 4. Stack
- LIFO
- Push/Pop → O(1)

---

## 5. Queue
- FIFO
- Enqueue/Dequeue → O(1)

---

## 6. Recursion
- Function calls itself
- Needs base case
- Uses stack memory

---

## 7. Searching
- Linear Search → O(n)
- Binary Search → O(log n)

---

## Key Takeaways
- Choose data structure based on problem
- Always think about time complexity
- Binary search is faster but needs sorted data
