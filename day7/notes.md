# Day 7 - Revision Day

## Topics Revised
- Arrays
- Linked List
- Stack
- Queue
- Recursion
- Time Complexity

---

## 1. Arrays
- Stored in contiguous memory
- Fast access → O(1)
- Insertion/Deletion → O(n)

---

## 2. Linked List
- Nodes connected using pointers
- Dynamic size
- Access → O(n)
- Insertion/Deletion → O(1) (if position known)

---

## 3. Stack
- LIFO (Last In First Out)
- Operations: Push, Pop, Peek
- All operations → O(1)

---

## 4. Queue
- FIFO (First In First Out)
- Operations: Enqueue, Dequeue
- All operations → O(1)

---

## 5. Recursion
- Function calls itself
- Needs base case
- Uses stack memory

---

## 6. Time Complexity
- O(1) → Constant
- O(n) → Linear
- O(log n) → Logarithmic
- O(n^2) → Quadratic

---

## Key Takeaways
- Choose correct data structure based on problem
- Always think about time complexity
- Strong basics are important for advanced topics
