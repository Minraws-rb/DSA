## 1. What is DSA?
DSA (Data Structures and Algorithms) helps in organizing data efficiently and solving problems optimally.


## 2. Time Complexity (Big-O)

It describes how fast an algorithm runs.

Common complexities:
- O(1) → Constant
- O(n) → Linear
- O(n^2) → Quadratic
- O(log n) → Logarithmic

Example:
Loop from 0 to n → O(n)



## 3. Arrays

- Collection of elements of same type
- Stored in contiguous memory
- Index starts from 0

Example:
int arr[5] = {1,2,3,4,5};

Access:
arr[0] → 1



## Key Takeaways
- Time complexity is very important in DSA
- Arrays are the simplest data structure
- Always think about optimization
